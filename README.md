# STAR WARS ENCYCLOPEDIA

> Create an app that lists Star Wars characters and allows you to search for them by name.

Requirements:

    - Use the API https://swapi.dev ✔️
    - The design is free choice. ✔️ 
    - Must show a list and a detail view of each character. ✔️
    - The list must be paginated. ✔️
    - The list must have a search filter. ✔️
    - When selecting a character from the list, the detail view will show relevant data. ✔️
    - OPTIONAL: Detail of movies, vehicles, starships, species... ❌
    - OPTIONAL: Photo of each character. ✔️
